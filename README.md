# 🐾 Patinhas Felizes

Projeto acadêmico desenvolvido para a disciplina de **Desenvolvimento Front-End**, ministrada pelo professor **Alexandre Cajazeiras**, no **Instituto Federal do Piauí (IFPI)**.

## 📖 Sobre o Projeto

O Patinhas Felizes é uma plataforma web criada com o objetivo de promover a adoção responsável e divulgar animais resgatados que procuram um novo lar.

O projeto foi desenvolvido como atividade prática da disciplina de Desenvolvimento Front-End, aplicando conceitos de estruturação de páginas web, estilização e organização de interfaces utilizando HTML e CSS.

## 🎯 Objetivos

* Incentivar a adoção responsável de animais;
* Divulgar animais disponíveis para adoção;
* Conscientizar sobre a importância da proteção animal;
* Aplicar conhecimentos de desenvolvimento web adquiridos em sala de aula;
* Desenvolver habilidades na construção de interfaces para a web.

## 🚀 Funcionalidades

* Página inicial de apresentação do projeto;
* Área dedicada à adoção de animais;
* Página com detalhes dos animais;
* Galeria de imagens;
* Seção com informações sobre como ajudar a causa;
* Navegação simples e intuitiva.

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Git
* GitHub
* GitHub Pages

## 📚 Conteúdos Aplicados

Durante o desenvolvimento deste projeto foram aplicados os seguintes conceitos:

* Estruturação de páginas utilizando HTML5;
* Utilização de elementos semânticos;
* Estilização de interfaces com CSS3;
* Organização de layouts e navegação;
* Boas práticas de desenvolvimento web;
* Controle de versão com Git;
* Publicação de sites utilizando GitHub Pages.
